# Проект: Документация и скриншоты

## 📁 Структура каталогов и файлов

### Документация
- `Attestacionnyi_list_PM11_dvukh...` — Аттестационный лист 
- `Dnevnik.docx` / `Dnevnik.pdf` — Дневник (в форматах Word и PDF)
- `Otchyotikikik.docx` / `Otchyotikikik.pdf` — Отчёт (также в двух форматах)
- `презентация426743.pptx` / `презентация426743.pdf` — Презентация (исходная PowerPoint и PDF-копия)

### Скриншоты
- `AD` — скриншоты/файлы, связанные с Active Directory
- `PO` — (скриншоты ПО или программного обеспечения)
- `Politica` — скриншоты политик (групповых, безопасности и т.п.)
- `TestNaSomvestimost` — скриншоты теста на совместимость
- `Users` — скриншоты управления пользователями

## 📝 Примечания
- Все файлы `*.docx` и `*.pptx` можно редактировать в Microsoft Office / LibreOffice.
- `*.pdf` версии предназначены для просмотра без изменений.
- Папки внутри `Скриншоты` могут содержать изображения (PNG, JPG) или дополнительные пояснительные файлы.

## 🚀 Назначение
Эта структура, вероятно, используется для:
- сдачи отчётности (аттестационный лист, дневник, отчёт)
- иллюстрации конфигурации системы (скриншоты AD, политик, тестов, пользователей)
- демонстрации совместимости ПО.

## 🛠 Как использовать
1. Открывайте документы в соответствующих приложениях.
2. Для просмотра скриншотов перейдите в нужную подпапку внутри `Скриншоты`.
3. При необходимости конвертируйте `.docx` → `.pdf` или наоборот.
